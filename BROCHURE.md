# ほめゴハン

**撮るだけで分かる。食べ方から、パフォーマンスを底上げ。**

---

## 🍽️ ほめゴハンとは

ほめゴハンは、**AI技術を活用した次世代の食事管理・献立提案プラットフォーム**です。

スポーツ・仕事・勉強でパフォーマンスを上げたい方が、食事を「写真で撮るだけ」で、AIが栄養バランスを推定・記録。その人の目的に合わせた**ほめコメント＋改善アドバイス＋週間献立**を自動生成します。

**プラットフォーム対応**: Web（PC・スマホブラウザ） / iOS / Android

---

## 💡 解決する課題

| 課題 | ほめゴハンのアプローチ |
|------|----------------------|
| **情報過多** | 栄養情報は多いが、自分の生活にどう活かすか分からない → **パーソナライズされた献立で解決** |
| **入力の手間** | 毎回の食事内容を記録する負担が大きい → **写真撮影だけでAI分析** |
| **心理的負担** | 「減らす・我慢する」方向のダイエットはストレス → **ポジティブなフィードバックで継続支援** |
| **方法論の欠如** | 何をどう食べればパフォーマンスが上がるか分からない → **科学的根拠に基づいたアドバイス** |

## ✨ 機能一覧（網羅版）

### 1. 食事記録・分析

#### 1-1. 食事写真AI分析
**写真を撮るだけで栄養を自動推定**

- 食事の写真を撮影するだけで、AIが料理を認識
- 複数の料理が写っていても個別に認識・分析
- **分析項目**:
  - カロリー (kcal)
  - タンパク質 (g)
  - 脂質 (g)
  - 炭水化物 (g)
  - 食物繊維 (g)
  - 塩分 (g)
  - ビタミン各種
  - ミネラル（カルシウム、鉄、マグネシウム等）

#### 1-2. 体重計写真分析
**体重計の表示を撮影して自動記録**

- 体重計の写真を撮るだけで数値を自動認識
- 手入力の手間を省いて記録継続をサポート

#### 1-3. 冷蔵庫分析
**冷蔵庫の中身を撮影して在庫把握**

- 冷蔵庫の写真から食材を自動認識
- 在庫に基づいた献立提案に活用
- 賞味期限が近い食材の優先利用を提案

#### 1-4. 写真分類
**撮影した写真を自動分類**

- 食事写真・体重計・健康機器などを自動判別
- 適切な分析機能に自動振り分け

#### 1-5. 手動入力
**写真がない場合の手動記録**

- 料理名から栄養情報を検索・入力
- 過去の記録から簡単コピー
- よく食べる料理のお気に入り登録

#### 1-6. 食事履歴
**過去の食事記録を振り返り**

- 日別・週別・月別での表示
- 栄養バランスの推移グラフ
- 特定の栄養素の過不足を可視化

### 2. 週間献立機能

#### 2-1. AI週間献立生成
**7日間×3食の献立を自動生成**

- **考慮される項目**:
  - ユーザーの栄養目標（カロリー・PFCバランス）
  - 健康状態・持病（高血圧、糖尿病など）
  - アレルギー・苦手食材
  - 好みの料理ジャンル（和食・洋食・中華など）
  - 調理経験レベル（初心者〜上級者）
  - 調理可能時間（平日・休日別）
  - キッチン設備（電子レンジ、オーブン等）
  - 家族人数

#### 2-2. 献立モード選択
**各食事で調理方法を選択可能**

| モード | 説明 |
|--------|------|
| **cook** | 自炊（レシピ付き献立を提案） |
| **quick** | 時短料理（15分以内で完成） |
| **buy** | 購入（コンビニ・スーパーの惣菜） |
| **out** | 外食（おすすめメニューを提案） |
| **skip** | 欠食（断食・ファスティング対応） |

#### 2-3. 1日献立変更モード
**特定の日だけ献立を変更**

- 1日単位での献立再生成
- 既存の週間献立との整合性を維持
- 急な予定変更に柔軟対応

#### 2-4. 個別献立再生成
**1食だけ別の献立に変更**

- 気に入らない献立を別の候補に差し替え
- 同じ栄養バランスを維持した代替案を提案
- 理由を指定して再生成（「もっと簡単に」「違う食材で」など）

#### 2-5. 献立テーマ設定
**日ごとのテーマを設定可能**

- 「スタミナ重視」「野菜たっぷり」「疲労回復」など
- チートデイの設定（週1回のご褒美デー）
- 栄養フォーカスの指定（タンパク質強化、減塩など）

#### 2-6. 作り置き対応
**週末まとめて作り置きプラン**

- 作り置きおかずの提案
- 複数日で使い回せるメニュー設計
- 保存期間を考慮した献立順序

### 3. 健康記録機能

#### 3-1. 基本健康データ
**日々の健康状態を記録**

| 項目 | 内容 |
|------|------|
| **体重** | kg単位で記録、グラフ表示 |
| **体脂肪率** | %で記録、体組成の推移 |
| **最高血圧** | mmHg（収縮期血圧） |
| **最低血圧** | mmHg（拡張期血圧） |
| **心拍数** | bpm（安静時心拍） |

#### 3-2. 生活習慣データ
**日常の生活習慣を記録**

| 項目 | 内容 |
|------|------|
| **睡眠時間** | 時間単位で記録 |
| **睡眠の質** | 5段階評価 |
| **気分スコア** | 5段階評価 |
| **ストレスレベル** | 5段階評価 |
| **体調スコア** | 5段階評価 |
| **水分摂取量** | ml単位で記録 |
| **歩数** | 歩数計・スマートウォッチ連携 |

#### 3-3. クイック記録
**よく使う項目だけサッと記録**

- 体重のみの簡易入力モード
- ワンタップで前回値をコピー
- ウィジェットからの直接入力

#### 3-4. 健康診断データ管理
**健康診断・検査結果を一元管理**

- 年次健康診断の結果を記録
- 血液検査データの管理
- 経年変化の可視化
- 健康診断写真からのOCR読み取り

#### 3-5. 健康目標設定
**達成したい目標を設定**

| 目標タイプ | 例 |
|-----------|-----|
| **体重目標** | 3ヶ月で5kg減量 |
| **体脂肪率目標** | 体脂肪率20%以下 |
| **歩数目標** | 毎日10,000歩 |
| **血圧目標** | 正常値を維持 |

- 目標達成度の進捗表示
- 残り期間と必要ペースの算出
- 達成時の通知・バッジ付与

#### 3-6. 健康グラフ
**データを視覚的に把握**

- 体重・体脂肪率の推移グラフ
- 血圧の変動グラフ
- 睡眠時間・質の推移
- 複数指標の相関表示
- 期間選択（週・月・年・全期間）

### 4. AI健康インサイト

#### 4-1. トレンド分析
**健康データの傾向を自動分析**

- 体重の増減傾向
- 血圧の変動パターン
- 睡眠の質の推移
- 季節・曜日による変動傾向

#### 4-2. 栄養×健康相関分析
**食事と健康の関係を発見**

- 「タンパク質摂取量が多い日は睡眠の質が良い」
- 「塩分が多いと翌日の血圧が高め」
- 「野菜摂取が少ない週は体調スコアが低下」

#### 4-3. パーソナルアドバイス
**あなただけの改善提案**

- 現在のデータに基づく具体的なアドバイス
- 目標達成に向けた行動提案
- 優先度付きの改善ポイント

#### 4-4. アラート機能
**異常値の早期検知**

- 急激な体重変動の検知
- 血圧の異常値警告
- 継続的な悪化傾向の通知
- 記録忘れのリマインド

#### 5-4. リスト共有

### 9. プロフィール・設定

#### 9-1. 基本情報
**パーソナライズの基礎データ**

| 項目 | 用途 |
|------|------|
| **ニックネーム** | 表示名 |
| **年齢** | 基礎代謝計算 |
| **性別** | 栄養目標計算 |
| **身長** | BMI計算 |
| **体重** | 基礎代謝計算 |
| **目標体重** | 目標設定 |

#### 9-2. フィットネス目標
**目的に応じた栄養設計**

| 目標 | 栄養バランス調整 |
|------|----------------|
| **減量** | カロリー制限、タンパク質増 |
| **筋力増強** | 高タンパク、適度な糖質 |
| **維持** | バランス重視 |
| **健康増進** | ビタミン・ミネラル重視 |
| **持久力向上** | 糖質増、適度な脂質 |

#### 9-3. 健康状態・持病
**健康上の制約を考慮**

- 高血圧 → 減塩献立
- 糖尿病 → 低GI食品優先
- 脂質異常症 → 脂質制限
- 腎臓病 → タンパク質・カリウム調整
- その他の持病への対応

#### 9-4. 食事制限・アレルギー
**食べられないものを登録**

- アレルギー（卵、乳、小麦、えび、かに、そば、落花生など）
- 苦手食材（ピーマン、セロリなど）
- 宗教上の制限（ハラール、コーシャなど）
- ベジタリアン・ヴィーガン対応

#### 9-5. 調理スキル・環境
**実現可能な献立を提案**

| 項目 | 選択肢 |
|------|--------|
| **調理経験** | 初心者/中級者/上級者 |
| **平日調理時間** | 15分/30分/60分/無制限 |
| **休日調理時間** | 30分/60分/120分/無制限 |
| **キッチン設備** | 電子レンジ/オーブン/炊飯器/圧力鍋 等 |

#### 9-6. 栄養目標設定
**カスタム栄養目標**

- 1日あたりカロリー目標
- PFC（タンパク質・脂質・炭水化物）バランス
- 個別栄養素の目標値（カルシウム、鉄など）
- 目標の自動計算 or 手動設定

#### 9-7. 通知設定
**リマインドのカスタマイズ**

- 食事記録リマインド（朝・昼・夕）
- 健康記録リマインド
- 献立確認リマインド
- バッジ獲得通知
- 週間レポート通知

#### 9-8. 家族・グループ設定
**複数人での利用**

- 家族メンバーの追加
- 献立人数の設定
- 買い物リストの共有設定

### 3ステップ分割アーキテクチャ（献立生成）

```
ステップ1: LLM生成
    └─→ 料理名・材料・手順を生成
    └─→ 自己トリガで次ステップへ

ステップ2: レビュー・修正
    └─→ 全体俯瞰で改善
    └─→ 被り回避・バランス調整
    └─→ 自己トリガで次ステップへ

ステップ3: 栄養計算・保存
    └─→ データセットから栄養値を計算
    └─→ DBに保存
    └─→ 完了通知
```

各ステップは独立した Edge Function として実行され、タイムアウトを回避しながら安定した生成を実現。

## 🌟 差別化ポイント

### 1. ポジティブなアプローチ
従来のダイエットアプリは「減らす・我慢する」方向でストレスが大きい。ほめゴハンは**「ほめコメント」と「次の一歩」**でポジティブに継続を支援。

### 2. 真のゼロコスト入力
写真を撮るだけ。カロリー入力や食材選択の手間をAIが完全に代行。

### 3. パフォーマンス志向
単なる「健康」ではなく、**スポーツ・仕事・勉強のパフォーマンス向上**にフォーカス。

### 4. エビデンスベースの栄養計算
LLMの推定値ではなく、**独自構築した食材データベース**に基づく科学的な栄養計算。

### 5. 3ステップ分割アーキテクチャ
長時間処理でもタイムアウトしない独自の分割実行方式で、安定した献立生成を実現。

### 6. マルチプラットフォーム
Web（PC・スマホブラウザ）、iOS、Androidのすべてに対応。デバイス間でデータ同期。

### 7. 組織対応
個人利用だけでなく、企業の健康経営、スポーツチーム、医療機関などの組織利用にも対応。
