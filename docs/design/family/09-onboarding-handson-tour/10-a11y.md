## 3. VoiceOver / TalkBack アナウンス文 (確定版)

各ステップ表示時に `AccessibilityInfo.announceForAccessibility()` で読み上げ:

| step | アナウンス文 |
|---|---|
| 0 | "ステップ 1 / 5。{nickname} さん、ようこそ。3 つの便利機能を一緒に試しましょう。" |
| 1.1 | "ステップ 2 / 5。写真から食事を追加します。" |
| 1.5 | "鶏の唐揚げ定食、780 キロカロリー。{nickname} さんの目標 {target_kcal} キロカロリー の {percent} パーセントです。" |
| 2.1 | "ステップ 3 / 5。AI で献立を作ります。" |
| 2.6 | "豚肉と野菜の生姜焼き、620 キロカロリー、調理時間 20 分。" |
| 3 | "ステップ 4 / 5。{nickname} さん、もう 2 つバッジを獲得しています。" |
| 4 | "ステップ 5 / 5。卒業おめでとう。使い方マスターのバッジを獲得しました。" |
| 5 | (welcome toast) "これからは自分のペースで使ってね、{nickname} さん。" |

### 3.1 アナウンス回数
- 各ステップで 1 回のみ (重複回避)
- `useEffect` で subStep 変化時に発火

```ts
useEffect(() => {
  const text = makeAnnouncement(currentStep, profile);
  AccessibilityInfo.announceForAccessibility(text);
}, [currentStep, profile.nickname]);
```

### 3.2 音声中断
- ユーザーが新しい操作 (タップ等) を行ったら現在のアナウンスは中断される (OS 仕様)
- 重要な情報は冒頭に置く (= "ステップ N / 5" を最初に)

## 4. キーボード操作 (Web)

### 4.1 フォーカス遷移

| キー | 動作 |
|---|---|
| Tab | フォーカス循環 (モーダル trap): 進捗 → 吹き出し本文 → 【次へ】 → 【スキップ】 (あれば) → 進捗 |
| Shift+Tab | 逆方向 |
| Enter / Space | 現在フォーカスのボタン押下 |
| Esc | 【スキップ】押下相当 (Step 0/4 のみ、Step 1-3 では無視) |

### 4.2 実装 (focus trap)

```tsx
import { useFocusTrap } from '@/hooks/useFocusTrap';

function TourOverlay(props: TourOverlayProps) {
  const trapRef = useFocusTrap();
  return <div ref={trapRef}>{/* ... */}</div>;
}
```

`useFocusTrap` は `react-focus-lock` 等で実装 (cross/03 既存 hook 流用 or 新規)。

### 4.3 初期フォーカス
- マウント時、【次へ】 (or 【はじめる】) に autoFocus
- スキップしか出ないステップ (Step 0/4) では【はじめる】 / 【ホームへ】 に autoFocus

## 5. 動的フォントサイズ対応

### 5.1 iOS Dynamic Type

| Type Size | 倍率 | レイアウト崩れ確認必須 |
|---|---|---|
| Default (Large) | 1.0 | 基準 |
| Extra Large (XL) | 1.15 | OK |
| Extra Extra Large (XXL) | 1.32 | OK |
| Extra Extra Extra Large (XXXL) | 1.55 | レイアウト調整必要 |
| Accessibility 1 (AX1) | 1.65 | スクロール必須 |
| Accessibility 5 (AX5) | 2.85 | 大幅調整 |

#### 実装
- `Text` コンポーネントは `allowFontScaling` を true (default)
- 吹き出し: `maxWidth: 280px` から flex に変更、内容に応じて折り返し
- 進捗ドットは固定 (8px、フォント連動なし)
- ボタン: 高さは `min-height: 44px` を維持しつつ、padding で文字に合わせて伸びる

### 5.2 Android Font scale

| Scale | 倍率 |
|---|---|
| Small | 0.85 |
| Default | 1.0 |
| Large | 1.15 |
| Largest (Extra Large) | 1.30 |

iOS と同じ調整方針。

### 5.3 Web (rem 基準)

- すべての font-size を `rem` で指定 (`text-sm` = `0.875rem`, `text-base` = `1rem`)
- ブラウザの「文字サイズ拡大」(Cmd/Ctrl + +) に追従

## 6. コントラスト比

### 6.1 WCAG AA 基準
- 通常テキスト: 4.5:1 以上
- 大テキスト (18pt+ or bold 14pt+): 3:1 以上
- UI コンポーネント / 図形: 3:1 以上 (non-text contrast)

### 6.2 実測値

| 要素 | 文字色 | 背景色 | 比率 | 判定 |
|---|---|---|---|---|
| 吹き出し本文 (light) | gray-900 (#111827) | white (#FFFFFF) | 17.4:1 | ✅ AA/AAA |
| 吹き出し本文 (dark) | gray-50 (#F9FAFB) | gray-900 (#111827) | 17.4:1 | ✅ AA/AAA |
| Step 0 タイトル | gray-900 | white | 17.4:1 | ✅ |
| Step 0 サブタイトル | gray-700 (#374151) | white | 9.7:1 | ✅ |
| 【はじめる】白文字 | white | primary-600 (#2563EB) | 4.7:1 | ✅ AA |
| 進捗ドット active | primary-600 | white | (UI、3:1 以上) | ✅ |
| 進捗ドット inactive | gray-300 (#D1D5DB) | white | (UI、3:1 以上) | ✅ |
| 【あとで】テキストボタン | gray-600 (#4B5563) | white | 7.6:1 | ✅ |
| ヒントテキスト ("タップで進む") | gray-500 (#6B7280) | white | 4.6:1 | ✅ AA (ギリギリ) |
| Spotlight 縁取り | primary-500 (#3B82F6) | dim 背景 | (UI、3:1 以上) | ✅ |

### 6.3 ダークモード

ダークモードでは:
- 吹き出し背景 gray-900 (#1F2937)
- 文字色 gray-50 (#F9FAFB)
- コントラスト 17:1 以上

`prefers-color-scheme: dark` (web) / `Appearance.getColorScheme()` (mobile) で自動切替。

## 7. 動きの低減 (`prefers-reduced-motion`)

### 7.1 検出方法

#### Web
```ts
const prefersReducedMotion = useMediaQuery('(prefers-reduced-motion: reduce)');
```

#### Mobile
```ts
import { AccessibilityInfo } from 'react-native';
const [reduceMotion, setReduceMotion] = useState(false);
useEffect(() => {
  AccessibilityInfo.isReduceMotionEnabled().then(setReduceMotion);
  const sub = AccessibilityInfo.addEventListener('reduceMotionChanged', setReduceMotion);
  return () => sub.remove();
}, []);
```

### 7.2 簡略化対応

| 要素 | 通常 | reduce-motion |
|---|---|---|
| Tour Overlay fade-in | 200ms | 0ms (即時) |
| Tour Overlay fade-out | 200ms | 0ms |
| Spotlight 移動 | 250ms | 0ms (jump cut) |
| 吹き出し fade-in | 150ms (delay 100ms) | 即時表示 |
| 進捗ドット切替 | 200ms | 即時 |
| Step 4 🎓 spring | 600ms | 即時 |
| Step 4 紙吹雪 | 表示 (3 秒ループ) | 非表示 |
| バッジカード scale | 300ms | 即時 |
| ローディングスピナー | 連続回転 | 連続回転 (機能上必要) |

## 10. 自動テスト (axe-core)

### 10.1 Web (Playwright + @axe-core/playwright)

```ts
import { test, expect } from '@playwright/test';
import AxeBuilder from '@axe-core/playwright';

test('handson-tour Step 0 a11y', async ({ page }) => {
  await page.goto('/handson-tour');
  await expect(page.getByTestId('tour-step-0')).toBeVisible();
  const results = await new AxeBuilder({ page })
    .withTags(['wcag2a', 'wcag2aa'])
    .analyze();
  expect(results.violations).toEqual([]);
});
```

各ステップで axe-core 違反 0 を担保。
